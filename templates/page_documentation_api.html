A4

Service permettant d'obtenir la liste des contraventions émises entre deux dates spécifiées.

/contrevenants get

get

Obtient la liste des contraventions entre deux dates spécifiées en paramètre. Les dates doivent être au format ISO 8601.

get /contrevenants

Obtient la liste des contraventions entre deux dates spécifiées en paramètre. Les dates doivent être au format ISO 8601.

  • Request
  • Response

Query Parameters

  • du: required(string)

    Date de début de la période de recherche au format YYYY-MM-DD.

    Example:

    2022-01-01
  • au: required(string)

    Date de fin de la période de recherche au format YYYY-MM-DD.

    Example:

    2022-12-31

HTTP status code 200

Resultat correct lorsque les dates de debut et de fin sont correctes

C1

Service REST permettant d'obtenir la liste des établissements ayant commis une ou plusieurs infractions. Pour chaque établissement, le service indique le nombre d'infractions connues. La liste est triée en ordre décroissant du nombre d'infractions.

/liste_etablissements get

get

Obtient la liste des établissements ayant commis des infractions, triée en ordre décroissant du nombre d'infractions.

get /liste_etablissements

Obtient la liste des établissements ayant commis des infractions, triée en ordre décroissant du nombre d'infractions.

  • Response

HTTP status code 200

Liste des établissements avec infractions triée en ordre décroissant du nombre d'infractions

Body

Media type: application/json

Type: any

Example:

[
  {
    "nom": "THALI CUISINE INDIENNE",
    "nb_infractions": 5
  },
  {
    "nom": "RELA MARCHE AFRICAIN",
    "nb_infractions": 3
  }
]

C2

Service REST permettant d'obtenir la liste des établissements ayant commis une ou plusieurs infractions au format XML. Pour chaque établissement, le service indique le nombre d'infractions connues. La liste est triée en ordre décroissant du nombre d'infractions.

/liste_etablissements_xml get

get

Obtient la liste des établissements ayant commis des infractions au format XML, triée en ordre décroissant du nombre d'infractions.

get /liste_etablissements_xml

Obtient la liste des établissements ayant commis des infractions au format XML, triée en ordre décroissant du nombre d'infractions.

  • Response

HTTP status code 200

Liste des établissements avec infractions triée en ordre décroissant du nombre d'infractions (format XML)

Body

Media type: application/xml

Type: any

Example:

<etablissements>
  <etablissement>
    <nom>THALI CUISINE INDIENNE</nom>
    <nb_infractions>5</nb_infractions>
  </etablissement>
  <etablissement>
    <nom>RELA MARCHE AFRICAIN</nom>
    <nb_infractions>3</nb_infractions>
  </etablissement>
  <!-- Autres établissements -->
</etablissements>

D1

Service permettant de soumettre une plainte concernant un établissement.

/soumettre_plainte post

post

Permet de soumettre une plainte concernant un établissement. Les données doivent être envoyées au format JSON et validées par un schéma JSON.

post /soumettre_plainte

Permet de soumettre une plainte concernant un établissement. Les données doivent être envoyées au format JSON et validées par un schéma JSON.

  • Request
  • Response

Body

Media type: application/json

Type: any

Example:

{
  "nom_etablissement": "RELA MARCHE AFRICAIN",
  "adresse": "925 Boul. Décarie, St-Laurent, Québec",
  "ville": "Montréal",
  "date_visite_client": "2024-04-15",
  "nom_client": "Ndiago",
  "prenom_client": "Meurie",
  "description_probleme": "Des cheveux dans la nourriture."

HTTP status code 400

post

  • Response
